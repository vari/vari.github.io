The Flying Stickman

A fly and shoot game optimized for different screen sizes and resolutions. Used OOP principles to model game entities and behaviours such as enemies, projectiles, and health. Debugged any bugs found through user testing via the Eclipse debugger after each update. Published on the Play store (under developer name A13).

  • Langauge: Android / Java
  • IDE: Eclipse
  • Date: June 2015 - January 2016
  • Play Store Link: The Flying Stickman
  • APK link: The Flying Stickman APK

Note: Due to several years of inactivity (and since I've forgotten the credentials for the account used to publish the game), the game has been removed from the Play store. You can find the APK file here if you'd like to give the game a try.

Blackjack

One player blackjack game with dealer AI and custom bets. Dealer AI conforms to blackjack rules using extensive decision structures and runtime variable values (i.e. the cards dealt). Designed to be robust, can handle invalid user input such as negative bets.

  • Langauge: VB.NET
  • IDE: Visual Studio
  • Date: May 2014 - June 2014

Concentration Game

Two player, turn based, concentration game with up to 15 cards. Players can enter their names and select the number of cards each round. Implemented complex algorithms to generate VB PictureBox objects dynamically each round, display random cards with no repeats, and parse player selected cards for match testing.

  • Langauge: VB.NET
  • IDE: Visual Studio
  • Date: April 2014 - May 2014

FF Enhancer

FF Enhancer is a Google Chrome extension for enhancing the filtering options on www.fanfiction.net. It allows you to define filters used to include/exclude stories based on story properties (word count, chapter count, etc.) and tags in the story descriptions.

  • Langauges: JavaScript, HTML, CSS
  • Date: December 2017 - Current (Worked on during free time)
  • Github: FF Enhancer
